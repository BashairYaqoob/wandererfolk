# 🌍 Wanderfolk – AI Travel Planner

Wanderfolk is an AI-powered travel planner that generates personalized travel itineraries based on user preferences.

Built using modern frontend technologies and AI, the application provides destination recommendations, day-wise itineraries, weather information, expense estimation, and trip management in a clean and immersive interface.

---

## ✨ Features

- 🤖 AI-generated travel itineraries
- 🌤 Live weather information
- 💰 Budget estimation
- 📅 Day-wise travel plans
- ❤️ Save and manage trips
- 📖 Travel journal
- 🗺 Destination explorer
- 📱 Fully responsive design
- 🎨 Modern editorial-style UI

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/wanderfolk.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

---

## 🔑 Environment Variables

Create a `.env` file.

```env
VITE_GROQ_API_KEY=your_api_key
```
